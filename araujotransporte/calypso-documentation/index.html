CSS Files and Structure

Backgrounds & Page Headers

How to change your background

Background images can be used only in boxed layout mode. Go to css/layout-boxed.css, line 1, and change the filename. Example, if you choose an image for the background called "ducks", instead of 

body {background:#fff url(../img/demo/16.jpg);}
change to 
body {background:#fff url(../img/demo/ducks.jpg);
Of course, we assume there already is a filename called ducks in img/demo/ folder. If not, upload it or change folder's path.

How to change your header image

Go to style.css and find the .pageheader-default class:

.pageheader-default {
	background:url(../img/demo/3.jpg);
}
Replace the same way as shown above for body background.

    Animations

    Animations supported by this theme: https://daneden.me/animate/

    How to use animations:

    Add the animation class to an existing class, in example let's add to col-md-6 class a fade up animation:

    fadeInUp
    If you want a load effect: 
    fadeInUpNow
    Load effects are available for fadeInUpNow, fadeInDownNow, fadeInLeftNow, fadeInRightNow, fadeInUpBigNow, FadeInDownBigNow, fadeInLeftBigNow, fadeInRightBigNow, slidea Please, see all available classes in style.css.

    It is not advisable to use the load effect for an element that appears on top of the page, the user will have to scroll down in order for that element to appear which is not quite nice

    To each animation add a "notransition" class. This will prevent the delay of the content on smaller devices.

    Example: 
    fadeInUpNow
